# Covenant Graph

**From legal text to live compliance.**

A category-defining enterprise loan intelligence platform that transforms syndicated loan agreements into a living, continuously updated covenant graph.

---

## Overview

Covenant Graph is not document storage. It's not reporting software. It's a **decision-support system** for syndicated loan markets that provides:

- **Graph-native visualization** of covenant relationships and dependencies
- **Amendment tracking** with visual diffs showing before/after states
- **Confidence-based compliance** assessment with full traceability
- **Bank-grade audit trails** for regulatory compliance

## Tech Stack

- **React 18** with TypeScript
- **Tailwind CSS** with custom design tokens
- **Framer Motion** for purposeful animations
- **React Router** for navigation
- **TanStack Query** for data management

## Design Philosophy

- **Calm & Authoritative**: No alarming colors or aggressive animations
- **Professional**: Built for hours-long use by financial professionals
- **Progressive Disclosure**: Complexity revealed on demand
- **Trust by Design**: Every decision traceable, every action auditable
